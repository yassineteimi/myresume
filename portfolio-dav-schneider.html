Cloud-Native Search Engine on Kubernetes Integrated with Salesforce

Description

  • Schneider Electric

  • August 2021, December 2021

Led the development of a cloud-native search engine application, deployed on Kubernetes, designed to streamline data retrieval across Salesforce and internal systems. This solution aimed to enhance search capabilities for business end-users, enabling rapid access to critical information such as product specifications, compliance reports, and customer interactions.

Main responsibilities

  • Led a cross-functional development team using Scrum, fostering collaboration between development, security, and operations departments to ensure smooth execution.
  • Designed and implemented a comprehensive CI/CD pipeline using GitHub Actions and Artifactory, enabling seamless deployment and faster iteration cycles.
  • Championed a DevSecOps culture, integrating tools like Aqua Security to embed security at every stage of the development lifecycle, effectively shifting security “left” and aligning with security-by-design principles.
  • Collaborated closely with Salesforce developers and business stakeholders to ensure seamless integration and alignment with business objectives.
  • Developed a high-level Kubernetes architecture that supported complex interactions with Salesforce, considering scalability, security, and operational efficiency.

Key achievements

Ò
  • Reduced build time by 30%, enhancing developer productivity and accelerating release cycles.
  • Improved software quality by 25%, using automated testing, code reviews, and continuous monitoring tools integrated into the CI/CD process.
  • Decreased critical security vulnerabilities (CVEs) by 40%, through proactive use of Aqua Security, ensuring compliance with industry-leading security standards.
  • Delivered a search engine capable of retrieving high-value operational data, empowering teams within Schneider Electric to access real-time information, from maintenance reports to customer records, driving informed business decisions.
